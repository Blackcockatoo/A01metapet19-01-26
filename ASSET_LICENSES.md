# Asset Licensing

This document defines the authoritative licensing terms for non-code assets in this repository.

## Default asset license

Unless otherwise stated below, visual, audio, and document assets in this repository are licensed under:

**Creative Commons Attribution 4.0 International (CC BY 4.0)**  
https://creativecommons.org/licenses/by/4.0/

Attribution should be provided as: "Meta-Pet project contributors" with a link to this repository.

## Asset categories and licenses

| Path / Category | License | Notes |
|---|---|---|
| `public/` static assets (`*.html`, `*.svg`, `*.json`) | CC BY 4.0 | Runtime-facing media and static experience files. |
| `sales-assets/` marketing screenshots, diagrams, and media placeholders | CC BY 4.0 | Replace placeholders with media that you have rights to publish under CC BY 4.0 or document exceptions here. |
| `docs/` diagrams, guides, and narrative documentation | CC BY 4.0 | Documentation text and diagrams are reusable with attribution. |
| `meta_pet_registry_demo/` demo PDFs and supporting artifacts | CC BY 4.0 | Demo/reference collateral for the project. |
| `zip 17.1.26/` bundled historical/reference artifacts | CC BY 4.0 | Legacy or reference materials distributed with this repo. |

## Third-party and external content exceptions

Some files are third-party material and keep their upstream license terms. These are **not** relicensed by CC BY 4.0:

| Path | Upstream source | Upstream license |
|---|---|---|
| `.expo/README.md` | Expo documentation | See Expo license terms: https://github.com/expo/expo/blob/main/LICENSE |
| `node_modules/` (if present locally) | NPM package authors | Per-package licenses declared in each dependency |

If you add third-party assets (fonts, icons, photos, audio, templates), add an entry here with source URL, author, and license.
