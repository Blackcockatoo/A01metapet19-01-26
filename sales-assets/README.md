## Screenshot Guidelines

### Recommended Dimensions
- **Hero images**: 1920x1080 (16:9) or 1242x2688 (iPhone Pro Max)
- **Feature screenshots**: 1280x720 minimum
- **GIFs**: 800x600, 10-15 seconds max

### Capture Tips
1. Run the app locally: `npm run dev`
2. Use browser dev tools to set consistent viewport
3. Capture at 2x resolution for retina displays
4. For GIFs, use screen recording software (LICEcap, ScreenToGif)
5. Ensure pet has good vitals for best animations
